# 测试数据目录

本目录存放用于测试的手写公式图像。

## 当前测试图像

目录中包含 46 张测试图像（1.png - 46.png），涵盖各种数学公式类型。

## 使用方法

### 方式一：GUI 界面

运行 GUI 程序，点击「选择图片」加载测试图像：

```bash
python gui.py
```

### 方式二：命令行

```bash
python main.py --image data/test/1.png
```

### 方式三：使用打包程序

直接运行 `dist/公式识别系统.exe`，然后选择测试图像。


## 添加新测试图像

将手写公式图像直接放入此目录，然后使用上述任一方式进行测试。
